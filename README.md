# Python-Repository
# Jarvis AI Assistant 🤖

A Python-based voice assistant that can listen to voice commands, speak responses, open websites, and perform basic automation tasks.

---

## 🚀 Features

- Voice recognition
- Text-to-speech
- Open websites using voice commands
- AI assistant interaction
- Continuous listening mode
- Wake word detection

---

## 🛠️ Technologies Used

- Python
- SpeechRecognition
- PyAudio
- pyttsx3

---

## 📦 Installation

Clone the repository:

```bash
git clone <your-github-repo-link>
```

Go to project folder:

```bash
cd jarvis-ai
```

Create virtual environment:

```bash
py -3.11 -m venv env1
```

Activate environment:

### Windows
```bash
.\env1\Scripts\Activate.ps1
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python main.py
```
